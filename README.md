PROJECT DESCRIPTION:

This project is about creating an ETL pipeline which extracts the data from Youtube Data API and transforms it using PySpark and loads the data in AWS S3.


USED TECHNOLOGIES, LIBRARIES AND APIs:

1. Apache Airflow 

2. Youtube Data API 

3. PySpark 

4. AWS S3 

5. Docker
